	CHANGELOG:

	*  minWidth attribute added to specify the minimum column width allowed
	
	*  Found and fixed small problem  in minimized version of colResizable v1.0 which 
	   did not allow applying the plugin to  more than one table if no ID was defined 
	   in their HTML - tags.
			
	*  The event onDrag was included

	*  Inner padding on cells was causing unexpected behaviors. It was fixed removing 
	   cells left and right padding programmatically.

	*  Other minor bugs

	Thanks to Martin Pruim (VvE-site.nl) and Yzlow for reporting bugs

	HISTORIAL DE CAMBIOS:

	*  Se agregó el atributo  minWidth para para permitir establecer el ancho  mínimo 
	   de las columnas
			
	*  Se solucionó pequeño  problema de minificación que  causaba que no se  pudiera 
	   aplicar el plugin a más de una tabla si esta no tenía definido un ID en su tag 
	   asociado del HTML.
			
	*  El evento onDrag  fue agregado para permitir capturar el tamaño de una columna 
	   en todo momento mientras se redimensiona.
		
	*  Se  detectó  y solucionó  problema  que  aparecía  al definir  padding lateral 
	   interno aplicado a las celdas, el cual provocaba comportamientos impredecibles
	   al redimensionar  las columnas. Se optó por eliminar dicho padding automática-
	   mente.
			
	*  Otros detalles menores

		
	Gracias a Martin Pruim (VvE-site.nl) y Yzlow por su ayuda reportando bugs.
